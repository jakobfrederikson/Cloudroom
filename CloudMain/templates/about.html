CS203 Investigative Studio 1


In this project we are tasked to create a Web Application of our choice and build a prototype based on the project proposal and inquiries in CS202


Cloudroom will streamline sharing files between the teachers and students, this is an online learning environment for teachers and students. Teachers can set up a Cloudroom and invite their students through a sharable link. Teachers will be able to create “assignments” or “homework” and view the submitted assignments, suggest fixes or mark the assignment etc.


A one-stop hub for any schoolwork to be done; it eliminates the hassle of going through google drive or searching through folders on your PC. Cloudroom can personalise your class experience by adding an inbuilt tool/feature to enhance the learning process of students i.e. For programming courses it will have a built-in code editor for Python, so students and teachers don’t have to download the necessary files to run Python.
